# Digital Behavior & Productivity Analytics

## Overview

Digital Behavior & Productivity Analytics is a data analytics project focused on understanding how behavioral factors such as sleep patterns, screen time, study/work hours, and stress levels influence productivity and burnout risk.

The project uses SQL-driven exploratory analysis and behavioral pattern discovery techniques to generate actionable insights from user behavior data. The analysis aims to identify productivity-driving habits, burnout indicators, and behavioral trends that can support data-driven decision making.

---

# Business Problem

Excessive digital usage, poor sleep habits, and unmanaged stress are increasingly affecting productivity and mental well-being.

This project analyzes user behavioral data to answer questions such as:

* Which behavioral patterns contribute to higher productivity?
* How does screen time influence stress and burnout?
* What habits are commonly observed among highly productive users?
* Are burnout patterns influenced more by behavior than occupation?

---

# Project Objectives

* Perform behavioral analytics using SQL
* Identify productivity and burnout trends
* Analyze relationships between sleep, screen time, stress, and productivity
* Segment users based on behavioral patterns
* Discover high-risk and high-performance user behaviors
* Generate insights for productivity and well-being analysis

---

# Dataset Information

The dataset contains behavioral and productivity-related attributes including:

| Column Name        | Description                       |
| ------------------ | --------------------------------- |
| User_ID            | Unique identifier for users       |
| Age                | User age                          |
| Occupation         | User occupation category          |
| Sleep_Hours        | Average daily sleep duration      |
| Study_hrs          | Daily study/work hours            |
| Screen_Time        | Daily screen exposure time        |
| Productivity_Score | Productivity performance score    |
| Stress_Level       | Stress category (Low/Medium/High) |
| Burnout_Risk       | Burnout risk level                |
